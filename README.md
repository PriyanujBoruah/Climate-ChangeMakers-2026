# AETHERIS: Hazard Monitoring & Prediction 🛡️

**Aetheris** is a real-time climate hazard monitoring tool developed for the **Climate Changemakers 2026** hackathon. It provides 24-hour predictive alerts for extreme weather events across 350+ global hubs, with a deep-density focus on Canadian municipalities.

## 🌟 Overview
Adaptation requires immediate action. This module "sifts" through real-time satellite and meteorological data to identify incoming hazards before they strike, providing local communities with the lead time necessary to protect life and infrastructure.

## 🚀 Key Features
- **Real-Time Sifting:** Monitors 350+ high-risk locations for Gale Winds, Flash Floods, and Severe Storms.
- **Local Analysis Engine:** Allows users to query specific regions for a detailed 7-day safety breakdown.
- **Predictive Risk Classification:** Automatically categorizes weather code intensities into actionable danger levels.
- **Canadian Density:** Includes 250+ specific locations across Ontario, Quebec, BC, Alberta, and the Maritimes.

## 🛠️ Tech Stack
- **Languages:** HTML5, CSS3 (Glassmorphism), JavaScript (ES6)
- **API:** Open-Meteo Forecast API

# AETHERIS: Snow & Heat Forecasting ❄️🔥

**Aetheris: Snow & Heat Forecasting** is a geospatial analytics dashboard that utilizes NASA satellite imagery to visualize systemic environmental shifts over a 10-year period.

## 🌟 Overview
To understand the future, we must visualize the past. This tool utilizes the **NASA GIBS API** to provide a synchronized, four-pane comparison of Land Surface Temperature and Snow Cover anomalies.

## 🚀 Key Features
- **Synchronized Quad-View:** Four Leaflet.js maps locked in a single viewport for perfect historical comparison.
- **Anomaly Engine:** Utilizes CSS `mix-blend-mode: difference` to visually highlight 10-year changes in heat and ice.
- **NASA MODIS Integration:** Real-time tiles from Terra/MODIS sensors for high-accuracy climate monitoring.
- **2030 Projection Lens:** A "Projection Mode" that amplifies current trends to visualize potential 2030 outcomes.

## 🛠️ Tech Stack
- **Map Engine:** Leaflet.js
- **Data Source:** NASA GIBS (Global Imagery Browse Services)
- **Filters:** CSS3 Visual Analytics Engine
